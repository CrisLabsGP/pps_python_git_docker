# pps_python_git_docker

# Título
- Bayeta de la fortuna

# Descripción 
- Aplicación web que consiste en que cada vez que accedamos a la web, nos dirá un texto auspicioso aleatorio.

# Instalación
## Instalar dependencias
- Una vez clonado el repositorio, ejecutar el siguiente comando:
``` cd ./pps_python_git_docker; pyhton3 -m pip install -r requirements.txt 

```

# Ejecutar la aplicación
- Ejecutar el siguiente comando:
```  python3 app.py 

```
